## Description
Webster is an editor that allows any user without design skills to easily create and process images.

Note: This project is archived and cannot be run due to outdated dependencies and loss of access to required services (e.g., database).
It is kept for demonstration purposes only, showcasing the code structure, design decisions, and problem-solving approach.

## Installation

```bash
# clone this project
$ git clone https://github.com/LizaNilova/webster.git

# install dependencies
$ make install
```

## Running the app

```bash
# start server
$ make start-server

# start client
$ make start-client
```
